
@@@> EplSite ETL

EplSite ETL is your own tool to create ETLs, doing extraction, 
transformation, validation, and load in a very fast and easy way.

It was the result of years of experience working on data migrations and BI(Data Warehousing), 

When you use EplSite ETL, you will see a lot of benefits in using this ETL, 
it does not need a lot of resources as most of the actual ETL software.

Features:

- Open Source.
- Easy to use.
- Low resource consuming.
- Just the necessary tools to do the job.
- Web interface.
- Very easy to customize it because it is developed in Perl, all the source code is included.
- It is possible to run transformations using cron jobs on Linux (When the scripts are exported).
